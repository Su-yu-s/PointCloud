Architecture

清晰分层强劲内核

检测引擎、点云 IO 与 Qt6 可视化层各司其职。C++17，CMake 一条命令编译。

核心库封装 RANSAC 算法与所有文件 IO 逻辑，GUI 层仅负责渲染与交互。模块边界清晰，易于测试、维护与扩展。

// 初始化检测器
auto detector = RansacDetector(Config::Architecture);

// 加载点云文件
auto cloud = PointCloudGenerator::loadFromFile("scene.pcd");

// 执行检测
auto shapes = detector.detect(cloud);
